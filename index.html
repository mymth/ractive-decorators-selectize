Ractive.js Selectize decorator plugin

download: ractive-decorators-selectize.js more plugins at ractivejs.org/plugins

This plugin is a decorator for Selectize.js inspired by @sandermarechal's Select2 plugin.

To customize selectize, set your options object to Ractive.decorators.selectize.types.default. You can also set up multiple configurations by adding another options object into Ractive.decorators.selectize.types as a new property.
